# Kurulum

1. Projeyi İndirmek

   - Git CLI <br>
     git clone https://github.com/beratcmn/word-addin.git

   - Website <br>
     ![](/public/ss1.png)

2. Projeyi Açmak

   - Gerekli kütüphaneleri yüklemek için: <br>
     `npm install`

   - Projeyi Webde çalıştırmak için: <br>
     `npm run web:test1`
